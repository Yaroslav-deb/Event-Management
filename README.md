# Event Management Full-stack System

Це фінальний проєкт комплексної системи управління подіями. Він поєднує REST та GraphQL API, авторизацію, роботу з базою даних MongoDB, а також клієнтську частину на React з інтерактивними графіками та чатом у реальному часі.

##  Структура проєкту
Проєкт розділений на дві частини:
- `/backend` — серверна частина (Node.js, Express, MongoDB, GraphQL, Socket.io).
- `/frontend` — клієнтська частина (React, Vite, Recharts, Socket.io-client).
